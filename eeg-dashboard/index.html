🎯 Project Overview

This project implements a state-of-the-art machine learning system for classifying mental states using EEG (electroencephalogram) signals. By analyzing brainwave patterns from 14 electrodes, our Random Forest model can predict eye state (open/closed) as a proxy for mental alertness with remarkable 98.86% accuracy.

🔬 Key Features:
  • ✨ Advanced signal preprocessing with bandpass filtering
  • 🧮 Feature engineering with temporal statistics
  • 🤖 Multiple ML algorithms comparison
  • 📊 Real-time prediction capabilities
  • 📈 Comprehensive performance visualization

ℹ️ About the Project

Learn about the EEG Mental State Classifier

🧠 EEG Mental State Classifier

This project represents a comprehensive implementation of a brain-computer interface system using electroencephalogram (EEG) signals to classify mental states. The system demonstrates advanced machine learning techniques applied to neuroscience data.

🎯 Project Goals

  • Develop a high-accuracy classifier for mental state detection
  • Create an educational platform for ML and BCI concepts
  • Demonstrate real-world applications of signal processing
  • Provide an interactive web interface for demonstrations

🔬 Technical Implementation

  • Data Source: UCI EEG Eye State Dataset
  • Signal Processing: Bandpass filtering, normalization
  • Feature Engineering: Rolling window statistics
  • Machine Learning: Random Forest with 98.86% accuracy
  • Web Framework: Django with Bootstrap frontend

🚀 Applications

This technology has applications in:

  • Medical monitoring and diagnosis
  • Driver alertness systems
  • Gaming and entertainment
  • Meditation and wellness tracking
  • Assistive technologies
