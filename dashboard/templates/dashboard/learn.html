🤖 Machine Learning Fundamentals

🎯 What is Machine Learning?

Machine Learning (ML) is a subset of artificial intelligence that enables computers to learn and make decisions from data without being explicitly programmed. Instead of writing specific instructions, we train algorithms to recognize patterns and make predictions.

📊 Supervised Learning

Learning from labeled examples to predict outcomes

Our Project Type
🔍 Unsupervised Learning

Finding hidden patterns in data without labels

🎮 Reinforcement Learning

Learning through trial and error with rewards

🔬 Our Project: Classification Problem

Our EEG mental state classifier is a binary classification problem. We're trying to predict one of two classes (Eyes Open vs Eyes Closed) based on EEG signal features.

📈 The Machine Learning Pipeline
  1. Data Collection: Gathered EEG signals from UCI repository
  2. Data Preprocessing: Cleaned and prepared the data
  3. Feature Engineering: Created meaningful features
  4. Model Training: Taught algorithms to recognize patterns
  5. Model Evaluation: Tested performance on unseen data
  6. Model Deployment: Made the model available in web app
💡 Key Concept: Train-Test Split

We split our data into:

  • Training Set (80%): Used to teach the model
  • Test Set (20%): Used to evaluate performance

This prevents overfitting!

Random Forest combines multiple decision trees. Think of it like asking 100 experts and taking the majority vote.

  • Robust: Less prone to overfitting than single trees
  • Feature Importance: Tells us which features matter most

  • Handles Mixed Data: Works with different types of features

🧠 Brain-Computer Interface (BCI)

🧠 What is a Brain-Computer Interface?

A Brain-Computer Interface (BCI) enables direct communication between the brain and an external device. Instead of using keyboards or mice, BCIs read brain signals to control computers.

⚡ How Brain Signals Work
  • Neurons: Brain cells that communicate using electrical signals
  • Neural Networks: Groups of connected neurons
  • Brain Waves: Patterns of electrical activity we can measure
  • Mental States: Different patterns create different states
🌊 Brain Wave Types
  • Delta (0.5-4 Hz): Deep sleep
  • Theta (4-8 Hz): Drowsiness, meditation
  • Alpha (8-13 Hz): Relaxed, eyes closed
  • Beta (13-30 Hz): Active thinking, eyes open
  • Gamma (30-50 Hz): High-level cognitive activity

📊 EEG: Electroencephalography

EEG measures electrical activity in the brain using electrodes placed on the scalp. It's like listening to millions of neurons working together.

🎯 Our Dataset: EEG Eye State
  • Source: UCI Machine Learning Repository
  • Participants: Volunteers wearing EEG headsets
  • Task: Simple - eyes open vs eyes closed
  • Duration: 117 seconds of continuous recording
  • Sampling Rate: 128 Hz (128 measurements per second)
🔬 Why Eyes Open vs Closed?

This is one of the most reliable signals in EEG! When you close your eyes, your brain shows distinct alpha wave patterns (8-13 Hz), especially in the occipital region (back of the head). It's perfect "hello world" BCI project!

🔧 Signal Processing Steps

1. Raw EEG Signal

Direct electrical measurements from scalp. Contains brain signals mixed with noise from eye movements, muscles, and electrical interference.

2. Filtering

Bandpass Filter (0.5-50 Hz)
Remove slow drift and electrical noise, keeping only brain frequencies.

signal.butter(4, [0.5, 50], 'band')
3. Normalization

Z-score Standardization
Transform data to have mean=0, std=1. Ensures all electrodes contribute equally.

StandardScaler().fit_transform()

🌐 Django Web Framework

🌐 What is Django?

Django is a high-level Python web framework that encourages rapid development and clean design. It's like having a complete toolkit - it handles the boring stuff so you can focus on your app.

🏗️ Django Architecture: MTV Pattern
  • Model: Handles data and database interactions
  • Template: Handles presentation (HTML)
  • View: Handles logic and connects Model & Template
🚀 Why Choose Django?
  • Batteries Included: Admin panel, authentication, ORM
  • Security: Built-in protection against attacks
  • Scalable: Used by Instagram, Pinterest, Mozilla
  • Fast Development: Don't repeat yourself (DRY)

🎯 How Our Dashboard Works

1. 🔗 URL Routing

When you visit http://localhost:8000/predict/, Django follows this path:

  1. eeg_dashboard/urls.py: Main router directs to dashboard app
  2. dashboard/urls.py: Maps '/predict/' to predict_mental_state view
  3. dashboard/views.py: predict_mental_state function handles the request
2. 👁️ Views (The Brain of Django)

Views contain the logic. Our predict view:

  • Handles GET requests (show the form)
  • Handles POST requests (process predictions)
  • Loads the trained ML model
  • Returns JSON responses
3. 🎨 Templates (The Face of Django)

Templates are HTML files with Django's template language:

  • Template Inheritance: base.html contains common layout
  • Template Variables: Display data from views
  • Template Tags: Control structures and logic
  • URL Reversal: Generate URLs dynamically
